# PWA Icons

Place your app icons here:

- `icon-192.png` — 192×192 px (required for Android PWA install prompt)
- `icon-512.png` — 512×512 px (required for splash screen)

You can generate these from any SVG/PNG using:
- https://realfavicongenerator.net
- https://maskable.app
- `sharp` npm package for programmatic generation

For development, any placeholder PNG will work.
